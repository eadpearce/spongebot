# spongebot  

spongebot is a Discord bot that repeats everything you prefix with !mock iN a mOcKiNg ToNe.  

## Setup  

install discord.js

```
yarn add discord.js
```

or

```
npm install --save discord.js
```

Add an auth.js file with your discord app bot user token like so:  

```
module.exports = {
  token: "YOUR_TOKEN"
}
```
